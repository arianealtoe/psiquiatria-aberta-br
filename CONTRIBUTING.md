# Contributing

Thank you for your interest in contributing to Psiquiatria Aberta BR.

## Ways to contribute
- Improve clarity and structure of educational materials
- Correct grammar or formatting
- Suggest references
- Add new topic summaries
- Improve organization of the repository

## Guidelines
- Keep content evidence-based and educational
- Write clearly and accessibly
- Avoid sharing identifiable patient information
- Prefer neutral, professional language
- Cite reliable references whenever possible

## Pull requests
Please open a pull request with:
- A short summary of your contribution
- The reason for the change
- Any references used, if applicable
